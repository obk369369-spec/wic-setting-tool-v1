WIC 7번 세팅 도구 — 도구 카드 관리

이 도구는 WIC 각 도구의 세팅 뼈대를 만들고, GitHub/Deno 업로드 주소를 자동으로 만들어 주는 화면형 세팅 도구다. 화면에 보이는 입력만으로 카드·주소·정리본을 관리한다.
① 업로드 환경 설정 (한 번만 입력)
예: https://github.com/obk369369-spec
예: https://dash.deno.com/projects
위 두 줄은 전체 공통값이다. 한 번 입력해 두면 아래에서 만드는 모든 도구 카드가 이 값을 이용해 업로드용 주소 텍스트를 자동 생성한다.
② 한 줄 지시로 도구 카드 생성
● 한 블록 = “도구 이름 / 목적 첫 줄” + 목적 추가 줄들
● 블록과 블록 사이를 “빈 줄 1칸 이상”으로 구분하면, 각 블록이 하나의 도구 카드로 생성된다.
● 도구 이름에 tool8-verb-helper 같이 슬러그가 포함되어 있으면 그 값을 그대로 주소에 사용한다.
③ 도구 카드 목록
④ 정리본 (도구 목록·업로드 상태 요약)
[WIC 7번 세팅 도구 — 도구 목록 요약] 카드를 만들면 여기 정리본이 자동 갱신된다.
이 텍스트는 Word/TXT/GitHub README 등에 복사해서 “어떤 도구가 어디까지 올라갔는지” 기록용으로 사용한다.
